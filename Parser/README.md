# How to Configure

## You will need:
* Re/Flex
* g++
* make
* cmake

## Instructions
* Step 1: `git clone https://github.com/alonzno/Quack-Compiler.git`
* Step 2: `cd Scanner`
* Step 3  `cmake .`
* Step 4: `make`
* Step 5: `./lexer <filename>`

This program tokenizes a stream of Quack code.  Output will be written to standard output
